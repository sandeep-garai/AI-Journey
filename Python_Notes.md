DAY 1

A class is a blueprint for a real world identity. It comprises of class variables, methods which can be static methods or regular methods.

Objects are the instance of the class, it defines the class

Constructor is the entry point of the class, it constructs the instance variable based on the values passed.

Inheritance is the property of class by which a class can inherit or acquire the properties, methods or class variable by inheriting it. It can also overwrite the methods as per its own needs.

API short for Application Programming Interface, is a feature by which the application can be invoked and get the information without understanding the underlying mechanism of it. It is mostly provided by the application for easy access.

JSON is JavaScript Object Notation, a data format in which the data is stored in key and value format. Each data has an unique key to find it. It is mostly used in webpages.

Virtual Environment is a secondary environment created to have different versions of the software for different needs. So a computer can have different environments for its different needs.
